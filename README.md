# A-frame Experiment 1: Terrain

## Description

Getting the hang for A-frame and more three.js development. Building a simple poly terrain.

<hr>

## Notes

To install the Node dependencies:
> npm install

To run locally:
> npm run

To run webpack dever server:
> npm run-dev
